# my-portfolio-website
Personal resume and portfolio website built with HTML, CSS, and JavaScript

## 🚀 Features
- Multi-page navigation (Home, About, Projects, Contact)
- Responsive design (mobile & desktop)
- Semantic HTML5 structure
- Interactive JavaScript features
- Multimedia integration (audio/video)
- Progressive enhancement (works without JS)

## 🛠 Tech Stack
- HTML5
- CSS3 (Flexbox & Grid)
- JavaScript (Vanilla)

## 📁 Project Structure
/project-root
- index.html
- about.html
- gallery.html
- contact.html
- css/styles.css
- js/scripts.js
- assets/

## 📚 What I Learned
- Building responsive layouts using Flexbox/Grid
- Writing clean and semantic HTML
- Adding interactivity with JavaScript
- Structuring front-end projects professionally
